The `esp_afe` component wraps the Espressif Audio Front End pipeline behind the
audio-processor interface used by `esp_audio_stack`.

It provides AEC, noise suppression, VAD, AGC and optional dual-mic Speech
Enhancement/BSS.

```yaml
esp_afe:
  id: afe_processor
  type: sr
  mode: low_cost
  aec_enabled: true
  ns_enabled: true
  vad_enabled: true

esp_audio_stack:
  id: audio_stack
  processor_id: afe_processor
  sample_rate: 48000
  output_sample_rate: 16000
```

## Configuration variables

- **id** (*Optional*, ID): Manually specify the processor ID.
- **type** (*Optional*, string): AFE profile, `sr`, `vc` or `fd`. Defaults to `sr`.
- **mode** (*Optional*, string): `low_cost` or `high_perf`. Defaults to `low_cost`.
- **input_format** (*Optional*, string): ESP-SR input layout. Defaults to `auto`.
- **aec_enabled** (*Optional*, boolean): Enable acoustic echo cancellation.
- **ns_enabled** (*Optional*, boolean): Enable noise suppression.
- **vad_enabled** (*Optional*, boolean): Enable voice activity detection.
- **agc_enabled** (*Optional*, boolean): Enable automatic gain control.
- **mic_num** (*Optional*, int): Number of microphone channels, `1` or `2`.
- **se_enabled** (*Optional*, boolean): Enable dual-mic Speech Enhancement/BSS.
- **memory_alloc_mode** (*Optional*, string): ESP-SR memory allocation strategy. Defaults to `more_psram`.
- **feed_buf_in_psram** (*Optional*, boolean): Place the feed scratch buffer in PSRAM. Defaults to `false`.
- **feed_ring_in_psram** (*Optional*, boolean): Place the feed staging ring in PSRAM. Defaults to `false`.
- **fetch_ring_in_psram** (*Optional*, boolean): Place the fetch output ring in PSRAM. Defaults to `false`.

## Actions

### `esp_afe.set_mode`

Switches the AFE type and mode at runtime.

```yaml
on_...:
  - esp_afe.set_mode:
      id: afe_processor
      type: fd
      mode: high_perf
```
