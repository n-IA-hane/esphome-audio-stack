The `esp_audio_stack` component is a full-duplex audio backend for ESP32-S3 and
ESP32-P4 voice devices with PSRAM. It owns the low-level audio path: I2S buses,
hardware codec control, DMA, sample-rate conversion, playback buffering and the
echo-cancellation reference.

It exposes normal ESPHome `microphone` and `speaker` platforms, so wake word,
Voice Assistant, media playback and custom call components can share one audio
backend.

This component requires the ESP-IDF framework and the `psram` component. Arduino
is not supported.

One resident audio task is created during setup and parks while idle. I2S
channels, DMA and codec paths start on first consumer and stop after the last
consumer leaves; the task TCB/stack remains allocated across those cycles.

```yaml
i2c:
  sda: GPIOXX
  scl: GPIOXX

esp_audio_stack:
  id: audio_stack
  i2s_bclk_pin: GPIOXX
  i2s_lrclk_pin: GPIOXX
  i2s_din_pin: GPIOXX
  i2s_dout_pin: GPIOXX
  codec:
    input:
      type: es8311
    output:
      type: es8311

microphone:
  - platform: esp_audio_stack
    id: stack_mic
    esp_audio_stack_id: audio_stack

speaker:
  - platform: esp_audio_stack
    id: stack_speaker
    esp_audio_stack_id: audio_stack
```

Set `sample_rate`, `output_sample_rate`, slot width and codec options when the
board requires values different from the defaults.

## Configuration variables

- **id** (*Optional*, ID): Manually specify the ID used for code generation.
- **sample_rate** (*Optional*, int): Physical I2S bus and speaker sample rate, from `8000` to `48000`. Defaults to `16000`.
- **output_sample_rate** (*Optional*, int): Microphone sample rate exposed to consumers, from `8000` to `48000`. Must divide `sample_rate` exactly, with a maximum decimation ratio of 6.
- **bits_per_sample** (*Optional*, int): Sample container on the bus. One of `16`, `24`, `32`. Defaults to `16`.
- **slot_bit_width** (*Optional*, string or int): Physical slot width on the bus, or `auto` to follow `bits_per_sample`. Defaults to `auto`.
- **num_channels** (*Optional*, int): Physical channel count on the standard I2S bus, `1` or `2`. Defaults to `1`.
- **speaker_channels** (*Optional*, int): Playback channel count, `1` or `2`. Defaults to `1`.
- **mic_channel** (*Optional*, string): Stereo slot carrying the microphone, `left` or `right`. Defaults to `left`.
- **rx_slot_mode** (*Optional*, string): Read one or both stereo slots on RX, `mono` or `stereo`. Defaults to `mono`.
- **tx_channel** (*Optional*, string): TX slot placement in mono-on-stereo layouts, `left` or `right`. Defaults to `left`.
- **correct_dc_offset** (*Optional*, boolean): Remove DC bias from the capture path. Defaults to `false`.
- **input_gain** (*Optional*, float): Digital gain applied before the processor, from `0.01` to `32.0`. Defaults to `1.0`.
- **master_volume_min_db** (*Optional*, float): Bottom of the volume curve in dB, from `-96.0` to `0.0`.

### Single-bus I2S

- **i2s_num** (*Optional*, int): I2S peripheral number. Defaults to `0`.
- **i2s_lrclk_pin** (**Required**, pin): Word-select/LRCLK pin unless `rx_bus` and `tx_bus` are used.
- **i2s_bclk_pin** (**Required**, pin): Bit-clock pin unless `rx_bus` and `tx_bus` are used.
- **i2s_mclk_pin** (*Optional*, pin): Master clock pin.
- **i2s_din_pin** (*Optional*, pin): Data-in pin.
- **i2s_dout_pin** (*Optional*, pin): Data-out pin.
- **i2s_mode** (*Optional*, string): Clock role, `primary` or `secondary`. Defaults to `primary`.
- **i2s_comm_fmt** (*Optional*, string): Frame format. One of `philips`, `msb`, `pcm_short`, `pcm_long`. Defaults to `philips`.
- **mclk_multiple** (*Optional*, int): MCLK to sample-rate ratio. One of `128`, `256`, `384`, `512`. Defaults to `256`.
- **use_apll** (*Optional*, boolean): Use the APLL clock source. In the maintained target set, only ESP32-P4 supports it. Defaults to `false`.
- **dma_desc_num** (*Optional*, int): DMA descriptor count, from `2` to `16`. Defaults to `6`.
- **dma_frame_num** (*Optional*, int): Frames per DMA descriptor, from `64` to `4092`.

### Dual-bus I2S

- **rx_bus** (*Optional*): Capture bus. Requires `i2s_num`, `i2s_lrclk_pin`, `i2s_bclk_pin` and `i2s_din_pin`.
- **tx_bus** (*Optional*): Playback bus. Requires `i2s_num`, `i2s_lrclk_pin`, `i2s_bclk_pin` and `i2s_dout_pin`.

`rx_bus` and `tx_bus` must be configured together on different I2S peripherals.
Dual-bus mode does not support TDM.

### Processor and reference

- **processor_id** (*Optional*, ID): ID of an `esp_aec` or `esp_afe` processor.
- **aec_reference** (*Optional*, string): Software playback-reference mode, `ring_buffer` or `previous_frame`. Defaults to `ring_buffer`.
- **aec_reference_buffer_ms** (*Optional*, int): Software reference ring capacity in milliseconds, from `32` to `500`. Defaults to `80`.
- **use_stereo_aec_reference** (*Optional*, boolean): Use stereo codec DAC feedback as the reference. Defaults to `false`.
- **reference_channel** (*Optional*, string): Reference channel, `left` or `right`. Defaults to `left`.
- **use_tdm_reference** (*Optional*, boolean): Use a TDM input slot as the hardware speaker reference. Defaults to `false`.
- **tdm_total_slots** (*Optional*, int): Total slots in the TDM frame, from `2` to `8`. Defaults to `4`.
- **tdm_mic_slot** (*Optional*, int): Zero-based microphone slot, from `0` to `7`. Defaults to `0`.
- **tdm_mic_slots** (*Optional*, list): One or two microphone slots for multi-microphone layouts.
- **tdm_ref_slot** (*Optional*, int): Slot carrying the speaker reference, from `0` to `7`. Defaults to `1`.
- **tdm_tx_slot** (*Optional*, int): Playback slot, from `0` to `7`. Defaults to `0`.

### Codec

- **codec** (*Optional*): Hardware codec configuration.
- **codec.input** (*Optional*): ADC-side codec. Supported types are `es7210`, `es8311`, `es8388`, `es8374` and `es8389`.
- **codec.output** (*Optional*): DAC-side codec. Supported types are `es8311`, `es8388`, `es8374` and `es8389`.
- Common codec fields include **address**, input **gain_db**, **use_mclk** and
  **no_dac_ref**. ES7210 additionally exposes **mic_selected**,
  **ref_channel** and **ref_gain_db**.

### Task, memory and diagnostics

- **task_priority** (*Optional*, int): Audio task priority, from `1` to `24`. Defaults to `19`.
- **task_core** (*Optional*, int): Core pinning, `-1` to `1`. Defaults to `0`.
- **task_stack_size** (*Optional*, int): Audio task stack in bytes, from `4096` to `32768`. Defaults to `8192`.
- **buffers_in_psram** (*Optional*, boolean): Place non-DMA audio buffers in PSRAM. Defaults to `false`.
- **audio_task_stack_in_psram** (*Optional*, boolean): Place the audio task stack in PSRAM through ESPHome's PSRAM task-stack helper. Defaults to `false`.
- **aec_ref_ring_in_psram** (*Optional*, boolean): Place the AEC reference ring in PSRAM. Defaults to `false`.
- **telemetry** (*Optional*, boolean): Enable per-stage diagnostics. Defaults to `false`.
- **telemetry_log_interval_frames** (*Optional*, int): Telemetry log cadence in frames, from `1` to `8192`. Defaults to `128`.
- **audio_effects** (*Optional*): Rate-converter tuning with
  **rate_cvt_complexity** (`1` to `3`) and **rate_cvt_perf_type** (`speed` or
  `memory`).

## Automation triggers

`on_start`, `on_idle` and `on_state` follow the aggregate runtime state.
`on_mic_start` / `on_mic_idle`, `on_speaker_start` / `on_speaker_idle`, and
`on_amplifier_required` / `on_amplifier_idle` expose consumer edges without a
timer-based lifecycle guess.

## Actions

### `esp_audio_stack.start`

Starts the audio path explicitly.

```yaml
on_...:
  - esp_audio_stack.start: audio_stack
```

### `esp_audio_stack.stop`

Requests a stop of the audio path.

### `esp_audio_stack.stop_and_wait`

Stops the audio path and blocks the automation until teardown completes.

```yaml
on_...:
  - esp_audio_stack.stop_and_wait: audio_stack
  - deep_sleep.enter: sleeper
```

## Condition

### `esp_audio_stack.is_idle`

Returns true when the stack is in the `idle` runtime state.
